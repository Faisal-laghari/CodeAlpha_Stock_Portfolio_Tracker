# CodeAlpha Stock Portfolio Tracker

Description

This project is a simple Stock Portfolio Tracker built in Python. It allows users to enter stock symbols and quantities, calculates the total investment value, and can save the results to a TXT or CSV file.

# Features

- Hardcoded stock price dictionary
- User input validation
- Portfolio summary table
- Total investment calculation
- Save results to TXT and CSV files

# Technologies Used

- Python

Concepts Used

- Dictionary
- Functions
- Loops
- File Handling
- Input and Output
- Error Handling

# How to Run

1. Open the project folder.
2. Run the command:

python Stock_Portfolio_Tracker.py
